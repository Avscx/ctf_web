
## 该小型留言系统②具有以下功能：
* 登陆模块、注册模块，用户能上传头像，若无则使用默认头像。
* 添加发言模块，评论模块，类似于`github`的`issue`	
* 权限划分，至少得有管理员和普通用户区别
	* 管理员对所有的发言和评论具有修改、删除权限
	* 普通用户只有修改、删除自己的发言、评论权限
* 用户能上传照片作为自己发表话题的辅助。
